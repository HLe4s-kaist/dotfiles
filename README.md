# Dotfiles

This is dot files for me.

## Install Dependency

This dotfiles now supports for ...  
```
nvim >= 0.11.0
tmux
```
but maybe this will support others later.  
Commands in `Quick Start` section will install tools listed above.

## Quick Start

To install this dotfiles on your system,

### Unix / Linux

on bash
```bash
git clone https://github.com/HLe4s-kaist/dotfiles.git ~/dotfiles
source ~/dotfiles/install.bash
```

on fish, zsh, chsh ...
